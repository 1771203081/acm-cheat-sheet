#C++版
-----------------
本书使用的是陈硕开源的[模板](https://github.com/chenshuo/typeset)。

##编译
Tex Live 下编译成功

`xelatex -synctex=1 -interaction=nonstopmode "ACM-cheatsheet".tex`

在windows下，还需要事先安装[5个字体](http://weibo.com/1663402687/zrSNhgTGK)
