#C++版
-----------------
虽然是号称`C++版`，不过书中的代码几乎都是用纯C写的，个别地方用了STL。原因如下：

1. 写ACM算法，完全不需要面向对象的特性，面向过程的语言已经足够了。
2. 用纯C，可以逼迫你重写很多平时习以为常的数据结构，锻炼基本功。
3. 我有代码洁癖。C++是一个很糟糕的语言，它是自底向上慢慢打补丁拼凑起来的一门语言，而不是自顶向下设计出来的，没有很好的宏观设计。
